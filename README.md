# Flash Cards

A very simple flash card tool.

## Setup

```bash
pip install .
```

## Running

```bash
python3 -m flash_cards.quiz
```

## Development Setup

```bash
# Set up a virtual env for dependencies.
python3 -m venv --prompt flash-cards .venv

# Activate the virtual env.
source .venv/bin/activate

# Install dependencies.
pip install --editable .
```
